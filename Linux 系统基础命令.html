Linux 文件基本属性

xxxxxxxxxx
ll或ls -l
显示一个文件的属性以及文件所属的用户和组
    例:在最前方出drwxr-xr-x 代表
    0位:代表文件的属性
                    当为[ d ]则是目录
                    当为[ - ]则是文件；
                    若是[ l ]则表示为链接文档(link file)；
                    若是[ b ]则表示为装置文件里面的可供储存的接口设备(可随机存取装置)；
                    若是[ c ]则表示为装置文件里面的串行端口设备，例如键Linux磁盘盘、鼠标(一次性读取装置)。
    
    1-9位:每3位为一组,且均为rwx
                    r:可读
                    w:可写
                    x:可执行
                    -:在某位置时 则代表没有此权限
        第1-3位确定属主（该文件的所有者）拥有该文件的权限
        第4-6位确定属组（所有者的同组用户）拥有该文件的权限
        第7-9位确定其他用户拥有该文件的权限

chmod：更改文件9个属性

xxxxxxxxxx
每种身份(owner/group/others)各自的三个权限(r/w/x)分数是需要累加的
r:4
w:2
x:1
​
chmod [-R] xyz 文件或目录
xyz : 就是刚刚提到的数字类型的权限属性，为 rwx 属性数值的相加
-R : 进行递归(recursive)的持续变更，亦即连同次目录下的所有文件都会变更
​
方式2:chmod u=rwx,g=rwx,o=rwx 文件名
    用 u, g, o 来代表三种身份的权限
​
方式3: chmod a-rwx 文件名
指定一个文件中的所有用户取消a-后的某个类型权限
                chmod u/g/o-rwx       -号换成+就可以加权限 
                三个类型的权限也可以单独删除某个类型的某个权限

umask

xxxxxxxxxx
umask rwx
用于设置默认文件和目录的权限
例:若默认权限文件为rwxr--r--
                            目录为r--r--r--
                            则:umask ----wx-wx
                            即:umask 033

cp(复制文件或目录)

xxxxxxxxxx
cp 文件名或文件目录 指定目标目录
cp -
        a:相当于 -pdr的效果
        d:若来源档为连结档的属性(link file)，则复制连结档属性而非文件本身
        i:若目标档(destination)已经存在时，在覆盖时会先询问动作的进行
        p:连同文件的属性一起复制过去，而非使用默认属性
        r:递归持续复制，用於目录的复制行为

rm

xxxxxxxxxx
rm 文件名
移除文件或目录
​
rm -
        f:就是 force 的意思，忽略不存在的文件，不会出现警告信息
        i:互动模式，在删除前会询问使用者是否动作
        r:递归删除！最常用在目录的删除了Linux 文件内容查看！这是非常危险的选项
        rf:强制删除整个目录,包含里面的子目录,并不提示

文件搜索命令find

xxxxxxxxxx
find 搜索范围 匹配条件
​
find 搜索范围 -name 文件名
例:find /ect -name init
精准搜索某文件
​
find 搜索范围 -name *文件名*
模糊搜索,只要包含文件名的文件都将被搜索出来
​
find 搜索范围 -name 文件名*
查找所有以文件名开头的文件
​
find 搜索范围 -name 文件名???
查找文件名开头后三个未知名称的文件
​
find 搜索范围 -iname 文件名
搜索时不区分大小写
​
注:不要在服务器高峰期时使用查找,尽量不使用,并且查找的范围越小越好,否则会严重占用性能
​
find 范围路径 -size +数字 
                                        -数字
                                        数字
    +:大于指定数字文件
    -:小于指定数字文件
    单纯的数字就是指定大小 
    
    find 范围路径 -user 所有者名称
    按照所有找查找,-group 根据所属组范围查找
    
    find 范围路径 查找类型 时间
                                -amin 访问时间
                                -cmin 文件属性
                                -mmin 文件内容
                例: find /etc -cmin -5
                在/etc 下查找5分钟内修改过文件属性的文件和目录
                
    locate 文件名
    可以快速查找到相应的文件  update可以更新locate库,否则有些文件找不到
                    -i 不区分大小写
                    
    grep 查找字符串 文件名或路径
    在文档内可以直接查找带有指定字符串的那一行,加-i 都可以区分大小写
     
    grep -v ^# 文件名
    排除指定字符串,排除#开头的行

压缩解压命令

xxxxxxxxxx
压缩格式:  .gz
gzip 文件名 
可以压缩成 ,gz格式的压缩文件,只能压缩文件不能压缩目录,并且不保留源原文件
​
gunzip 文件吗
可以解压.gz的压缩文件
​
打包:tar
    tar -vcf  压缩后文件名 目录
    打包之后才可以将目录压缩成.gz
    
    tar -zcf 压缩后文件名 目录
    可以打包后直接压缩成.gz
    
    tar -zxf .tar.gz文件
    解包并解压文件
    
    压缩格式:zip
    zip 压缩后的文件名 被压缩文件名
    保留原文件,加-r 并且可以压缩目录
    
    unzip
    解压.zip压缩文件
     
    bzip2 -k 被压缩文件
    可以保留原文件,压缩比 比较大,可以压缩较大文件
    
    tar -cjf 压缩后文件名 被压缩文件
    可以生成打包并.tar.bz2的压缩文件
     
    bunzip2 -k 文件名
    解压bz2的压缩文件,加-k可以保留压缩包 
    
    tar -xjf .tar.bz2文件
    可以解包并解压.tar.bz2压缩文件
